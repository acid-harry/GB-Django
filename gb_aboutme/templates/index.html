Добро пожаловать в CRUD функционал Django

Ниже представлены операции CRUD, которые вы можете выполнить:

Создание

Для создания нового клиента, продукта или заказа выполните POST запрос по следующим эндпоинтам:

  • /create_client/
  • /create_product/
  • /create_order/

Чтение

Для получения информации о клиенте, продукте или заказе выполните GET запрос по следующим эндпоинтам:

  • /client/{client_id}/
  • /product/{product_id}/
  • /order/{order_id}/

Обновление

Для обновления данных клиента, продукта или заказа выполните POST запрос с обновленными данными по следующим эндпоинтам:

  • /update_client/{client_id}/
  • /update_product/{product_id}/
  • /update_order/{order_id}/

Удаление

Для удаления клиента, продукта или заказа выполните POST запрос по следующим эндпоинтам:

  • /delete_client/{client_id}/
  • /delete_product/{product_id}/
  • /delete_order/{order_id}/

Список заказанных товаров

Для получения списка товаров, заказанных за определенный период, выполните GET запрос с указанием числа дней по следующему эндпоинту:

  • /ordered_products/{days}/
